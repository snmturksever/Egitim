HTML

İşaretleme ve etiketleme dili olarak tanımlanmaktadır. Web sitesi iskeleti oluşturur.İstisnalar hariç başlangıç etiketi ile kapanması en önemli koşuludur.Başlık, paragraf, link, görsel,sıralı/sırasız listeler, tablo ve formlardan oluşur.

En çok Kullanılan Html Etiketleri

  • ! --> Basitçe yapı oluşturur
  • h1 --> Başlık etiketi
  • p --> Paragraf etiketi
  • br --> Bir satır boşluk bırakır
  • a href --> Link ekleme etiketi
  • ol ve ul --> Sıralı/sırasız liste
  • strong --> Kalın punto oluşturur
  • script --> Javascript kodlarını HTML içerisinde yazmayı sağlar
  • button --> Buton oluşturmaya yarar
  • tittle --> Başlık belirtmek için kullanılır
  • style --> Biçimlendirme için kullanılır
  • script src= --> Etiketlerde senkron çalışmayı sağlar
  • link rel =-->

Emmet / Zen Codding

  • + --> Kardeş öğe oluşturur
  • > --> İçinde öğe oluşturur
  • ^ --> Yukarı çık öğe oluştur
  • * --> Kere öğe aç
  • .class --> Sınıf oluşturmak için kullanılır
  • #id --> ID/ Tanım oluşturur

CSS için kullanılan etiketler

  • span --> işaretleme amacı ile kullanılır
  • div --> blok oluşturur

Sementik Html etiketleri blocklandırma için kolaylık sağlar.Her yerde div etiketinin kullanımı karışıklığa sebep olmaması için fayda sağlar.
